# Kitchen Bar Specifications

**Updated:** 2026-01-08

---

## Specifications

| What                                | Measurement         | Notes                                 |
| ----------------------------------- | ------------------- | ------------------------------------- |
| **Step height**                     | 23.5"               | Fixed — can't change this             |
| **Bar length**                      | 90"                 | Fits 3-4 people comfortably           |
| **Bar top depth**                   | 30"                 | Total counter depth                   |
| **Bar top**                         | Butcher block, 1.5" | Chevron pattern preferred             |
| **Bar height (from kitchen floor)** | 40"                 | Counter height + step = 63.5" from LR |
| **Cabinet body height**             | 34.5"               | Standard base cabinet height          |
| **Toe kick height**                 | 4"                  | Standard proportion                   |
| **Cabinet depth**                   | 14"                 | Doors face living room                |
| **Overhang toward LR**              | 3"                  | Past cabinet face                     |
| **Knee space (kitchen side)**       | 12"                 | For bar stool seating                 |

---
