# Task Completion Checklist

When completing a task in this project:

## 1. Verify Changes
- [ ] Run `npm run dev` and visually check the visualization
- [ ] Test any interactive controls (dropdowns, presets)
- [ ] Verify dimensions and calculations are correct

## 2. Code Quality
- [ ] Maintain consistent code style (no semicolons, 2-space indent)
- [ ] Keep inline styles consistent with existing patterns
- [ ] Comment complex calculations or design decisions

## 3. Documentation
- [ ] Update `docs/design-spec.md` if design parameters change
- [ ] Move old versions to `archive/` if creating new version

## 4. Git (only if user requests)
- [ ] `git status` to review changes
- [ ] `git add .` to stage
- [ ] `git commit -m "descriptive message"` to commit
