## Phase 1: Fabrication - Cutting

**Duration:** 8-10 hours
**When:** Weekend 1, Day 1

### Reference Documents

| Document | What to Use It For |
|----------|-------------------|
| **WS4: MASTER-CUT-LIST.md** | Exact dimensions for every cut |
| **WS4: plywood-layout.md** | Optimal cutting sequence from sheets |
| **WS4: cutting-sequence.md** | Order of operations |
| **WS3: ADDENDUM-no-table-saw.md** | Store rip strategy, circular saw techniques |

### Tasks

1. **Store Ripping (at purchase)**
   - 3/4" plywood to 14" strips
   - 3/4" plywood to 34.5" strips
   - 1/2" plywood to 10" strips

2. **Home Cutting Sequence**
   - Set up straightedge guide for circular saw
   - Cut all 14" depth pieces (cabinet panels)
   - Cut all 30" depth pieces (counter panels)
   - Cut all 4" height pieces (toe kick)
   - Cut door and panel pieces
   - Cut drawer box parts

3. **Quality Check**
   - Verify all critical dimensions within 1/16"
   - Stack matching pieces to verify identical
   - Label all pieces immediately

## Phase 2: Fabrication - Drilling & Prep

**Duration:** 3-4 hours
**When:** Weekend 1, Day 1 (afternoon)

### Reference Documents

| Document | What to Use It For |
|----------|-------------------|
| **WS4: MASTER-CUT-LIST.md** | Pocket hole locations, wire hole specs |
| **WS3: BUILD-SEQUENCE-FINAL.md** | Pocket hole settings and technique |
| **WS7: wire-routing.md** | Electrical hole locations (H1-H6) |

### Tasks

1. **Pocket Hole Drilling**
   - Set jig for 3/4" material
   - Drill all cabinet panel pocket holes
   - Switch to 1/2" setting
   - Drill all drawer pocket holes

2. **Electrical Prep Holes**
   - H1: 1" hole, left side, 6" from bottom
   - H2: 1/2" hole, left side, 4" from bottom
   - H5: 3/4" hole, divider, 6" from bottom
   - Deburr all holes

3. **Surface Preparation**
   - Sand all pieces: 120, 150, 220 grit progression
   - Do NOT sand glue joint areas
   - Ease all visible edges

## Phase 3: Pre-Finishing

**Duration:** 2-3 hours
**When:** Weekend 1, Day 2

### Reference Documents

| Document | What to Use It For |
|----------|-------------------|
| **WS6: FINISHING-GUIDE.md** | Primer selection, application technique |
| **WS6: finishing-sequence.md** | What to pre-finish, timing |
| **WS3: BUILD-SEQUENCE-FINAL.md** | Pre-finish strategy section |

### Tasks

## Phase 4: Cabinet Box Assembly

**Duration:** 4-6 hours
**When:** Weekend 2, Day 1

### Reference Documents

| Document | What to Use It For |
|----------|-------------------|
| **WS3: BUILD-SEQUENCE-FINAL.md** | Step-by-step assembly order |
| **WS3: CONSTRUCTION-BEST-PRACTICES.md** | Square verification, clamping |
| **WS4: MASTER-CUT-LIST.md** | Section widths for divider placement |

### Tasks

1. **Build Toe Kick Platform**
   - Assemble 2x4 frame
   - Verify square (diagonals equal)
   - Add cross supports

2. **Install Dividers to Bottom**
   - Mark positions: 30", 60", 92"
   - Glue and screw dividers
   - Check plumb on each

3. **Attach Cabinet Top**
   - Glue top edges of dividers
   - Position and align
   - Drive pocket screws
   - CHECK SQUARE - diagonals must match

4. **Install Living Room Panel**
   - Apply glue to all edges
   - Brad nail every 6"
   - Nail into dividers

**Duration:** 2-3 hours
**When:** Weekend 2 (during cabinet glue cure)

### Reference Documents

| Document | What to Use It For |

### Tasks

3. **Let Cure**

## Phase 7: Finishing (Paint & Stain)

**Duration:** 8-10 hours (spread over multiple days)
**When:** Weekend 3

### Reference Documents

| Document | What to Use It For |
|----------|-------------------|
| **WS6: FINISHING-GUIDE.md** | Complete paint application technique |
| **WS6: paint-colors.md** | Color selection and matching |
| **WS6: finishing-sequence.md** | Day-by-day timeline |
| **WS3: BUILD-SEQUENCE-FINAL.md** | Parallel work opportunities |

### Tasks

**Day 1: Prep and First Coats**
1. Final sand (220 grit) on all cabinet surfaces
2. Vacuum and tack cloth
3. Apply primer to remaining unprimed surfaces
4. Wait 2 hours
5. Apply first coat paint

**Day 2: Second Coats**
1. Light sand with 320 grit
2. Tack cloth
3. Apply second coat paint
4. Apply clear coat to stained pieces

**Day 3: Touch-up and Doors**
1. Touch up any issues
2. Prime and paint drawer fronts
3. Prime and paint pre-made doors

## Phase 8: Hardware Installation

**Duration:** 4-5 hours
**When:** Weekend 3-4 (after paint cure)

### Reference Documents

| Document | What to Use It For |
|----------|-------------------|
| **WS5: HARDWARE-RECOMMENDATION.md** | All hardware specifications |
| **WS5: barn-door-hardware.md** | Barn door track installation |
| **WS5: hinges-and-pulls.md** | Hinge boring, pull placement |
| **WS3: BUILD-SEQUENCE-FINAL.md** | Hardware installation section |

### Tasks

1. **Drawer Slide Installation**
   - Mark slide heights (all must match)
   - Install cabinet-side slides
   - Install drawer-side slides
   - Test each drawer

2. **Cabinet Hinge Installation**
   - Drill 35mm cup holes
   - Install hinges in doors
   - Attach mounting plates to cabinet
   - Hang and adjust doors

3. **Barn Door Track Installation**
   - Mark level line 2" above cabinet
   - Pre-drill and mount track
   - Hang door
   - Install floor guide and soft-close

4. **Pull Installation**
   - Make drilling template
   - Mark all holes
   - Drill and install all pulls

## Phase 9: Final Installation

**Duration:** 3-4 hours
**When:** Weekend 4

### Reference Documents

| Document | What to Use It For |
|----------|-------------------|

### Tasks

## Phase 10: Electrical & Final Finishing

**Duration:** 3-4 hours
**When:** Weekend 4+ (after countertop installed)

### Reference Documents

| Document | What to Use It For |
|----------|-------------------|
| **WS7: ELECTRICAL-PLAN.md** | Complete electrical system |
| **WS7: outlet-options.md** | Pop-up outlet installation |
| **WS7: led-lighting.md** | LED strip installation |
| **WS7: foot-rail.md** | Foot rail mounting |
| **WS6: butcher-block-finish.md** | Rubio Monocoat application |

### Tasks

1. **Pop-Up Outlet Installation**
   - Cut 3-3/4" hole in countertop
   - Run Romex from junction box
   - Wire and install outlet
   - Test GFCI

2. **LED Lighting Installation**
   - Mount power supply behind drawer
   - Install aluminum channels
   - Stick LED strips in channels
   - Connect wiring
   - Install dimmer and motion sensor

3. **Foot Rail Installation**
   - Cut pipe to 92" length
   - Deburr ends
   - Mount brackets to toe kick (into backing blocks)
   - Install rail

4. **Butcher Block Finishing**
   - Apply Rubio Monocoat
   - Wipe off excess
   - Allow 5-7 days cure

5. **Final Touch-ups**
   - Touch up any paint damage
   - Adjust all hardware
   - Final cleaning

## Parallel Work Opportunities

### While Primer Dries (2 hours)
- Drill electrical pathway holes
- Cut X-frame trim pieces

### While First Paint Coat Dries (4 hours)
- Build drawer boxes
- Build barn door X-frame

### While Glue Cures (30+ minutes)
- Cut next batch of parts
- Sand finished pieces

### While Drawer Glue Cures (overnight)
- Build X-frame panels
- Apply stain to wood accents

### While Second Paint Coat Dries (4 hours)
- Test-fit drawer slides on scrap
- Prepare hardware for installation

### While Final Paint Cures (24 hours)
- Install barn door track (not door yet)
- Wire electrical pathways
