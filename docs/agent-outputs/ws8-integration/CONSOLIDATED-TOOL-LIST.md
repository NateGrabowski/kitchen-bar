## Overview

This is the master tool list for the Kitchen Bar Cabinet project. All tools from all work streams have been consolidated and categorized by necessity level.

**IMPORTANT:** This build is designed for a builder with NO TABLE SAW. All cutting operations use:
- Big box store panel ripping service
- Miter saw
- Circular saw with straightedge guide

## TOOL QUALITY RECOMMENDATIONS

### Worth Investing In (Buy Quality)

| Tool | Why Quality Matters | Recommended Brands |
|------|---------------------|-------------------|
| **Clamps** | Last a lifetime, critical for accuracy | Bessey, Jorgensen |
| **Pocket Hole Jig** | Precision = strong joints | Kreg (not knockoffs) |
| **Measuring Tools** | Inaccurate = bad cuts | Empire, Irwin, Swanson |
| **Brushes** | Cheap = brush marks | Purdy, Wooster |

| Tool | Why Budget is Fine | Suggested Sources |

| **Drill Bits** | Wear items | Harbor Freight OK |
| **Spring Clamps** | Low stress use | Any brand |
| **Drop Cloths** | Single use often | Cheapest available |

## TOOL MAINTENANCE NOTES

### Before Build

- [ ] Verify miter saw blade is sharp and cuts clean
- [ ] Check circular saw blade - replace if teeth damaged
- [ ] Test pocket hole jig on scrap - verify bit depth
- [ ] Charge all cordless tool batteries
- [ ] Verify clamp mechanisms work smoothly
- [ ] Check level accuracy on known flat surface
- [ ] Clean sander pad, stock sandpaper discs

### During Build

- [ ] Clean pocket hole jig after each session
- [ ] Wipe glue off clamps immediately
- [ ] Keep saw blades free of pitch buildup
- [ ] Replace sanding discs when worn

### After Build

- [ ] Clean all tools
- [ ] Organize and store properly
- [ ] Note any tools that need repair/replacement
