# Tutorial 1: Woodshop Diaries - DIY Kitchen Cabinets From Plywood

**Date Accessed:** 2026-01-09

## Overview

Woodshop Diaries presents a complete method for building kitchen cabinets using only plywood, without the need for planing, joining, or milling solid hardwood. This tutorial is particularly valuable for DIYers who want professional results with minimal tools.

## Key Construction Method

**Frameless Cabinet Construction** - The tutorial focuses on building frameless (European-style) cabinets with full overlay doors, which:
- Eliminates the need for face frames
- Requires only plywood (no solid hardwood needed)
- Simplifies the build process
- Maximizes interior cabinet space

## Materials

### Primary Materials
- **¾" Birch PureBond Plywood** - Used for all cabinet carcasses
  - Cabinet-grade plywood with hardwood veneer (birch, maple, or oak)
  - Comes pre-sanded and smooth
  - No additional surface prep needed
- **¼" Plywood** - For cabinet backs
- **Edge Banding** - To finish exposed plywood edges

### Hardware
- Pocket hole screws
- Wood glue
- Soft-close hinges
- Drawer slides (soft-close recommended)

## Tools Required

### Essential Tools (Minimum)
- **Circular saw** with cutting guides - For breaking down full sheets
- **Pocket hole jig** (Kreg or similar)
- **Drill/driver**
- **Measuring tape and square**
- **Clamps**

### Recommended Tools (Makes it easier)
- Table saw
- Miter saw
- Brad nailer
- Router for edge details

## Skill Level

**Beginner to Intermediate** - The tutorial is specifically designed to be accessible to DIYers who:
- Have basic woodworking skills
- Are comfortable using power tools
- Can follow detailed plans
- Have patience for precise measuring

## Key Techniques Taught

### 1. Plywood Selection
- Choose cabinet-grade (hardwood) plywood
- Inspect sheets for defects before purchasing
- Plan cuts to maximize sheet usage and grain direction

### 2. Breaking Down Plywood Sheets
- Use circular saw with straightedge guide
- Cut full sheets into manageable pieces first
- Work from largest pieces to smallest

### 3. Box Construction with Pocket Holes
- Drill pocket holes on interior faces
- Apply wood glue to joints
- Assemble with pocket screws
- Check for square during assembly

### 4. Frameless Design Benefits
- No face frame means fewer parts
- Full overlay doors cover plywood edges
- Simpler joinery requirements
- Modern, clean appearance

### 5. Cabinet Back Installation
- Use ¼" plywood backs
- Square up cabinet boxes using the back
- Attach with brad nails or staples

## Main Takeaways for Our Project

### Why This Method Works for a 14" Deep Kitchen Bar

1. **Plywood is dimensionally stable** - Won't warp or expand like solid wood, critical for shallow 14" cabinets

2. **Frameless construction saves depth** - With limited 14" depth, eliminating the face frame gives us more interior space

3. **Minimal tool requirements** - You already have most needed tools (table saw, miter saw, router)

4. **Cost effective** - Building with plywood is significantly cheaper than solid hardwood while still achieving quality results

5. **Time efficient** - One to two ¾" plywood sheets can make 2+ cabinet boxes, speeding up production

### Specific Applications to Kitchen Bar Project

**Applies directly:**
- Base cabinet construction at 34.5" height matches our spec
- Pocket hole joinery is fast and strong for cabinet boxes
- Birch plywood can be painted for modern farmhouse aesthetic

**Needs adaptation:**
- Tutorial assumes standard 24" depth; we need to scale to 14" depth
- Face frames may still be desirable on visible kitchen side for Shaker style
- Living room side (back) will be fully visible, needs premium finish

### Estimated Build Time

According to Woodshop Diaries approach:
- **Cabinet boxes:** 2-4 hours per box (first one takes longest)
- **For our 92" bar:** Approximately 3 cabinet boxes = 8-12 hours total
- **With experience:** Time decreases significantly after first cabinet

## Critical Success Factors

1. **Accurate measurements** - Plywood doesn't forgive measurement errors
2. **Sharp blades** - Prevents tearout on veneer faces
3. **Test cuts** - Always make test cuts on scrap for critical dimensions
4. **Dry fit first** - Assemble without glue to check fit before final assembly
5. **Square as you go** - Check square at every assembly step
