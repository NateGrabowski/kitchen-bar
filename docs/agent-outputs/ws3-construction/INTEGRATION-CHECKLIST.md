# Integration Checklist

**Comprehensive Verification for Kitchen Bar Cabinet Build**

Use this checklist throughout the build to ensure nothing is forgotten. Each section must be completed before proceeding to the next phase.

---

## Phase 0: Pre-Build Verification

### Materials Ordered/Purchased

**Hardware (Order First - Shipping Takes 3-5 Days):**
- [ ] Drawer slides: 3 pairs, 14" full-extension soft-close (e.g., Liberty D80618C)
- [ ] Barn door hardware: 72" track kit with soft-close (e.g., WINSOON)
- [ ] European cup hinges: 3 pairs, soft-close, 110-degree
- [ ] Drawer pulls: 3 (matching style)
- [ ] Door pulls/knobs: 3 (matching style)
- [ ] Barn door pull: 1 cup pull or flush pull
- [ ] Screws: #6 x 1-1/4" for cabinet, #6 x 3/4" for drawers

**Lumber:**
- [ ] 3/4" plywood: 3 sheets (store-ripped to spec widths)
- [ ] 1/2" plywood: 1 sheet (for drawer boxes)
- [ ] 1/4" plywood: 1 sheet (for drawer bottoms, toe kick facing)
- [ ] 1x4 boards: 30 linear feet (for X-frame trim)

**Finishing Supplies:**
- [ ] Primer: 1 gallon shellac-based or quality latex
- [ ] Cabinet paint: 1 gallon, satin or semi-gloss
- [ ] Wood glue: 16 oz minimum
- [ ] Wood filler: 1 tube
- [ ] Sandpaper: 120, 150, 220, 320 grit
- [ ] Painter's tape: 2 rolls
- [ ] Tack cloths: 6 pack

**Fasteners:**
- [ ] Pocket screws 1-1/4" coarse: 1 box (100+)
- [ ] Pocket screws 3/4" fine: 1 box (100+)
- [ ] Brad nails 1-1/4": 1 box
- [ ] Finish nails 2": small box (backup)

**Electrical (Per WS7 Plan):**
- [ ] USB outlet: 1
- [ ] LED strip lights: per plan
- [ ] Wire: per plan
- [ ] Grommets: 2-3
- [ ] Wire staples or clips

### Tools Verified Working

**Power Tools:**
- [ ] Circular saw with 40+ tooth blade
- [ ] Miter saw (tested for 90° and 45° accuracy)
- [ ] Drill/driver with charged batteries (2 batteries recommended)
- [ ] Random orbital sander with fresh pads
- [ ] Brad nailer with air/battery (or hammer for manual nailing)
- [ ] Router with straight bit (for dados, if using)

**Hand Tools:**
- [ ] Kreg pocket hole jig (drill bit sharp)
- [ ] 4' level
- [ ] Framing square
- [ ] Tape measure (25')
- [ ] Combination square
- [ ] Hammer
- [ ] Clamps: 4+ bar clamps, 8+ spring clamps

**Accessories:**
- [ ] Circular saw cutting guide / straightedge
- [ ] Sawhorses (2 pairs recommended)
- [ ] Shop vacuum
- [ ] Safety glasses
- [ ] Ear protection
- [ ] Dust mask (N95)

## Phase 1: Cutting Verification

### Before Starting Cuts

- [ ] All plywood inspected for damage, delamination
- [ ] Store rip cuts verified at correct widths
- [ ] Cutting guide tested on scrap for accuracy
- [ ] Miter saw blade square to fence (test cut + measure)
- [ ] Workspace cleared, sawhorses stable

### After All Cuts Complete

**Cabinet Components:**
- [ ] Cabinet bottom: 14" x 92" (verify with tape measure)
- [ ] Cabinet top: 14" x 92" (must match bottom exactly)
- [ ] Dividers (3): 14" x 34.5" (all identical height)
- [ ] Shelves (as needed): 14" x opening width
- [ ] Toe kick front: 4" x 92"
- [ ] Toe kick sides: 4" x depth

**Door/Panel Components:**
- [ ] Living room panel: 34.5" x 92"
- [ ] Barn door panel: 30" x 35"
- [ ] Double door panels: per final measurements
- [ ] X-frame trim pieces: per measurements

**Drawer Components:**
- [ ] Drawer fronts (3): 29.5" x 10" x 1/2"
- [ ] Drawer backs (3): 29.5" x 10" x 1/2"
- [ ] Drawer sides (6): 13" x 10" x 1/2"
- [ ] Drawer bottoms (3): 29" x 12.5" x 1/4" (or 29.5" x 13" x 1/2" if pocket screw method)

**Labeling:**
- [ ] All pieces labeled (part name, orientation)
- [ ] Inside faces marked (for pocket hole drilling)
- [ ] Matching pieces grouped together

**Dimension Verification:**
- [ ] Critical pieces within 1/16" of specification
- [ ] Matching pieces stacked and edges compared
- [ ] No visible tear-out on visible edges

---

## Phase 2: Pre-Assembly Preparation

### Pocket Holes

**3/4" Material:**
- [ ] Jig set to 3/4" position
- [ ] Collar set slightly shallow (between 1/2" and 3/4")
- [ ] Test hole in scrap - no blowout
- [ ] Cabinet bottom: holes on TOP face, around perimeter
- [ ] Cabinet top: holes on BOTTOM face, around perimeter
- [ ] Dividers: holes on BOTH ends
- [ ] All holes at least 2" from edges

**1/2" Material:**
- [ ] Jig reset for 1/2" material
- [ ] Collar adjusted appropriately
- [ ] Test hole in scrap
- [ ] Drawer fronts: 3 holes each side edge (interior face)
- [ ] Drawer backs: 3 holes each side edge (interior face)
- [ ] Drawer sides: 3 holes along bottom (if not using dado)

### Electrical Preparation

**CRITICAL: Must complete before cabinet assembly**

- [ ] Wire pathway holes marked on dividers (per WS7 plan)
- [ ] Wire pathway holes drilled (1-1/4" diameter minimum)
- [ ] Grommet locations marked
- [ ] Grommet holes drilled (match grommet size)
- [ ] USB outlet rough-in location marked
- [ ] All holes deburred (smooth edges)
- [ ] Holes will not interfere with shelf placement

### Surface Preparation (Sanding)

- [ ] All visible faces sanded with 120 grit
- [ ] All visible faces sanded with 150 grit
- [ ] All visible faces sanded with 220 grit
- [ ] All sharp edges eased (one pass with sanding block)
- [ ] Dust removed (vacuum + tack cloth)
- [ ] NO sanding on glue joint surfaces (bare wood)

### Pre-Finishing Interior Surfaces

**Mask BEFORE painting:**
- [ ] All pocket hole areas masked (1" around each)
- [ ] All glue joint edges masked
- [ ] Tape pressed firmly (no paint bleed)

**Prime:**
- [ ] Cabinet bottom: TOP surface primed
- [ ] Cabinet top: BOTTOM surface primed
- [ ] Dividers: BOTH sides primed
- [ ] Living room panel: BACK surface primed (faces inside)
- [ ] Primer dried minimum 2 hours

---

## Phase 3: Cabinet Box Assembly

### Toe Kick Platform

- [ ] Toe kick pieces assembled (front + sides + supports)
- [ ] Platform is rectangular (diagonals equal)
- [ ] Platform is flat (check with straightedge)
- [ ] Set aside for later leveling

### Cabinet Box Assembly

**Dividers to Bottom:**
- [ ] Divider positions marked on cabinet bottom (per plan)
- [ ] Glue applied to divider bottom edges
- [ ] First divider positioned and clamped
- [ ] First divider checked for plumb (level)
- [ ] Pocket screws driven (do not overtighten)
- [ ] Process repeated for all dividers
- [ ] All dividers plumb and at correct positions

**Cabinet Top:**
- [ ] Glue applied to divider top edges
- [ ] Top positioned (pocket holes facing down)
- [ ] Top aligned flush with bottom edges all around
- [ ] Pocket screws driven

### QUALITY GATE: Box Square Verification

**STOP. Complete ALL checks before proceeding.**

- [ ] Overall dimensions: 92" L x 34.5" H x 14" D (within 1/8")
- [ ] All four corners checked with framing square
- [ ] Diagonal 1 measured (corner to corner)
- [ ] Diagonal 2 measured (opposite corners)
- [ ] Diagonals equal within 1/4"
- [ ] All dividers plumb when checked with level
- [ ] Cabinet bottom flat (straightedge test)
- [ ] No gaps at any joints

**If not square:**
- [ ] Problem identified
- [ ] Corrective action taken (clamp, shim, reassemble)
- [ ] Verification repeated
- [ ] Square confirmed before proceeding

### Living Room Panel

- [ ] Cabinet box verified square (see above)
- [ ] Cabinet positioned on back (kitchen side down)
- [ ] Glue applied around all box edges
- [ ] Living room panel positioned flush with box edges
- [ ] Brad nails driven every 6" around perimeter
- [ ] Brad nails driven into each divider
- [ ] Panel firmly attached (no flex or gaps)

---

## Phase 4: Sub-Assembly Verification

### Drawer Boxes (Complete All 3)

**Per Drawer:**
- [ ] Dry fit completed first (no glue)
- [ ] Sides attached to front (glue + 3 pocket screws per side)
- [ ] Bottom inserted in dado OR positioned for pocket screws
- [ ] Back attached (glue + 3 pocket screws per side)
- [ ] Bottom secured (if using pocket screw method)
- [ ] Diagonals measured and equal
- [ ] Glue squeeze-out wiped
- [ ] Set aside to cure (30 min minimum)

**All Drawers:**
- [ ] All 3 drawer boxes complete
- [ ] All 3 drawer boxes square
- [ ] Drawer boxes match each other (stack and compare)

### X-Frame Panels

**Per Panel:**
- [ ] Perimeter frame cut with 45° miters
- [ ] Frame dry-fit to panel (miters meet cleanly)
- [ ] Frame glued and brad-nailed to panel
- [ ] X-diagonal pieces measured in place
- [ ] X-diagonals cut to fit
- [ ] X-diagonals glued and brad-nailed
- [ ] Nails not in areas where hardware will be installed

### Barn Door

- [ ] Panel cut to 30" x 35"
- [ ] Panel square (diagonals equal)
- [ ] Perimeter frame attached
- [ ] X-pattern trim attached
- [ ] All trim secure
- [ ] Edges smooth
- [ ] Ready for finishing

---

## Phase 5: Finishing Verification

### Surface Prep Before Paint

- [ ] All sub-assemblies complete and cured
- [ ] All surfaces sanded with 220 grit
- [ ] All holes filled with wood filler (if visible)
- [ ] Filler sanded smooth
- [ ] All dust removed (vacuum + tack cloth)
- [ ] All glue joints masked with painter's tape

### Primer Application

- [ ] All previously unprimed surfaces primed
- [ ] Coverage even (no thin spots)
- [ ] Drip-free
- [ ] Dry time: 2 hours minimum

### First Paint Coat

- [ ] Light sand with 320 grit if needed
- [ ] Dust removed
- [ ] First coat applied evenly
- [ ] All surfaces covered
- [ ] Drip-free
- [ ] Dry time: 4 hours minimum

### Second Paint Coat

- [ ] First coat lightly sanded (320 grit)
- [ ] Dust removed (tack cloth)
- [ ] Second coat applied evenly
- [ ] Complete coverage verified
- [ ] Drip-free
- [ ] Final cure time: 24-48 hours before hardware

### Pre-Hardware Verification

- [ ] Paint fully cured (hard to touch, no tackiness)
- [ ] All surfaces smooth
- [ ] No drips, runs, or sags
- [ ] Color consistent across all components
- [ ] Touch-ups applied to any imperfections

---

## Phase 6: Hardware Installation

### Drawer Slide - Cabinet Side

**Per Drawer Opening:**
- [ ] Height position marked (center of drawer opening)
- [ ] Horizontal line drawn with level
- [ ] Cabinet slide positioned flush with front
- [ ] Pilot holes drilled
- [ ] Screws driven
- [ ] Slide checked for level
- [ ] Opposite side installed at EXACT same height
- [ ] Both sides verified at same height (measure from bottom)

**All Drawer Positions:**
- [ ] Bottom drawer slides installed and level
- [ ] Middle drawer slides installed and level
- [ ] Top drawer slides installed and level

### Drawer Slide - Drawer Side

**Per Drawer Box:**
- [ ] Drawer slide component positioned on box
- [ ] Flush with front of drawer
- [ ] Centered vertically on drawer side
- [ ] Pilot holes drilled
- [ ] Screws driven
- [ ] Opposite side at EXACT same position
- [ ] Both sides verified identical

### Drawer Installation

**Per Drawer:**
- [ ] Cabinet slides extended fully
- [ ] Drawer tilted and inserted
- [ ] Drawer components engaged with cabinet slides
- [ ] Drawer pushed in to test operation
- [ ] Drawer opens smoothly
- [ ] Drawer closes smoothly
- [ ] Soft-close engages properly
- [ ] Drawer level when closed
- [ ] No binding or sticking

### Barn Door Track Installation

- [ ] Living room panel verified structurally sound
- [ ] Track position marked (2" above cabinet top)
- [ ] Level line drawn full track length
- [ ] Track positioned on line
- [ ] Pilot holes drilled through track into panel
- [ ] Screws driven
- [ ] Track verified LEVEL with level
- [ ] Track does not flex or move

### Barn Door Hanging

- [ ] Roller/hanger brackets attached to door top
- [ ] Brackets at correct spacing per kit instructions
- [ ] Rollers spin freely
- [ ] Door lifted and rollers engaged with track
- [ ] Door hangs plumb (vertical)
- [ ] Door slides left freely
- [ ] Door slides right freely
- [ ] Door covers opening when closed
- [ ] Door clears opening when open
- [ ] Floor guide installed
- [ ] Soft-close mechanism installed
- [ ] Soft-close engages smoothly

### Hinge Installation

**Per Door:**
- [ ] Hinge positions marked (3" from top, 3" from bottom typical)
- [ ] Cup holes drilled with 35mm Forstner bit
- [ ] Hinge inserted in cup
- [ ] Mounting plate attached to cabinet
- [ ] Hinge clipped to plate
- [ ] Door operation tested
- [ ] Alignment adjusted (vertical, horizontal, depth)
- [ ] Consistent gaps all around door

## Phase 7: Drawer Front Installation

### Applied Front Method

**Per Drawer:**
- [ ] Drawer in cabinet and functioning
- [ ] 1/16" spacer strips positioned around opening
- [ ] Drawer front positioned on drawer box face
- [ ] Spacers maintain consistent gap
- [ ] Front centered on box
- [ ] Front marked or taped in position
- [ ] Pilot holes drilled from INSIDE drawer through box into front
- [ ] Temporary screws driven (2)
- [ ] Alignment verified
- [ ] Additional screws driven (4-6 total)
- [ ] Spacers removed
- [ ] Gaps verified consistent

### Handle Installation

- [ ] Handle position determined (typically centered)
- [ ] Drilling template created or used
- [ ] Hole positions marked on ALL drawer fronts
- [ ] Holes drilled
- [ ] Handles installed
- [ ] Handles level and consistent

## Phase 8: Electrical Integration

### Wire Routing

- [ ] Cabinet positioned in final location
- [ ] Wire access from power source identified
- [ ] Wire routed through pre-drilled pathways
- [ ] Wire secured with clips/staples (no loose wires)
- [ ] Grommets installed in cord pass-through holes
- [ ] Wire slack managed (no kinks, no excess visible)

### USB Outlet Installation

- [ ] Location accessible but hidden
- [ ] Outlet mounted securely
- [ ] Wiring connected per code
- [ ] Outlet tested functional
- [ ] Outlet cover installed

### LED Strip Installation

- [ ] Location per plan
- [ ] Surface cleaned before adhesive
- [ ] Strip applied straight
- [ ] Wiring connected
- [ ] Transformer/driver secured
- [ ] Lights tested functional
- [ ] Switch accessible and functional

### Foot Rail (If Installing)

- [ ] Backing block installed behind cabinet face
- [ ] Backing block securely attached
- [ ] Rail mounts positioned
- [ ] Pilot holes drilled through cabinet into backing
- [ ] Rail mounted
- [ ] Rail level
- [ ] Rail secure (no wobble)

## Phase 9: Final Installation

### Toe Kick Platform Installation

- [ ] Platform positioned in final location
- [ ] Platform leveled (shim as needed)
- [ ] All corners at same height
- [ ] Shims secured (won't move)
- [ ] Platform attached to floor if required

### Cabinet Installation

- [ ] Cabinet lifted onto toe kick platform
- [ ] Cabinet aligned with platform edges
- [ ] Cabinet secured to toe kick
- [ ] Cabinet level side-to-side
- [ ] Cabinet level front-to-back

### Toe Kick Facing

- [ ] Facing material cut to height
- [ ] Facing painted to match
- [ ] Facing positioned on toe kick frame
- [ ] Gaps consistent at top

- [ ] Nail holes filled and touched up
