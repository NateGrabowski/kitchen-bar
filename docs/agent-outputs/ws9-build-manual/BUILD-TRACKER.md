## Phase 6: Finishing

**Target Date:** __________
**Actual Date:** __________
**Time Spent:** __________

### Day 1: Sand and Prepare

- [ ] Sand all cabinet surfaces (150, then 220 grit)
- [ ] Sand all drawer fronts
- [ ] Sand cabinet doors
- [ ] Fill any holes, gaps, or defects
- [ ] Let filler dry
- [ ] Sand filler flush
- [ ] Vacuum all dust
- [ ] Wipe with tack cloth
- [ ] Wipe with denatured alcohol

### Day 2: Prime

- [ ] Set up finishing area (ventilated, 50-85F)
- [ ] Stir primer thoroughly
- [ ] Prime all cabinet surfaces (brush edges, roll flats)
- [ ] Prime drawer fronts
- [ ] Prime cabinet doors
- [ ] Let dry 45 minutes
- [ ] Light sand with 320 grit
- [ ] Vacuum and tack cloth

### Day 3: First Paint Coat

- [ ] Stir paint thoroughly
- [ ] Apply first coat (brush and roll technique)
- [ ] Let dry 16 hours (BM Advance)
- [ ] Light sand with 320 grit
- [ ] Vacuum and tack cloth

### Day 4: Final Paint Coat

- [ ] Apply final coat
- [ ] Let dry 24-48 hours before handling
- [ ] Allow full cure (7-14 days) before heavy use

### Days 5-7: Stain Wood Accents (If Applicable)

- [ ] Sand barn door X-frame trim (120, 150, 220)
- [ ] Apply pre-stain conditioner
- [ ] Apply Golden Oak stain
- [ ] Wipe off excess, let dry 8 hours
- [ ] Apply first coat polyurethane
- [ ] Let dry 3-4 hours
- [ ] Sand lightly with 320
- [ ] Apply second coat polyurethane
- [ ] Let dry 3-4 hours
- [ ] Sand lightly, apply third coat (optional)
- [ ] Let cure 24+ hours

### Quality Gate

Before proceeding to Phase 7:
- [ ] Paint smooth, no brush marks or drips
- [ ] Color consistent across all surfaces
- [ ] Stained wood has even color
- [ ] Clear coat smooth and protected
- [ ] All nail holes filled
- [ ] Full cure time noted

## Phase 8: Countertop

**Target Date:** __________
**Actual Date:** __________
**Time Spent:** __________

### Prepare Butcher Block

- [ ] Inspect slabs for damage
- [ ] Let acclimate 48-72 hours if not already done
- [ ] Plan joint location (offset from center)

### Join Slabs

- [ ] Position slabs face-down
- [ ] Align joining edges
- [ ] Mark for pocket holes (or biscuits)
- [ ] Drill pocket holes (or cut biscuit slots)
- [ ] Apply glue to both edges
- [ ] Join slabs with screws (or biscuits)
- [ ] Clamp across joint
- [ ] Let cure 24 hours

### Cut to Final Size

- [ ] Mark final dimensions (92" x 30")
- [ ] Cut with circular saw and straightedge
- [ ] Sand cut edges smooth

### Cut Pop-Up Outlet Hole

- [ ] Mark position: 15" from left, 6" from kitchen edge
- [ ] Use 3-3/4" hole saw
- [ ] Cut from top side
- [ ] Sand hole edges

### Install Countertop

- [ ] Position on cabinet
- [ ] Center side-to-side
- [ ] Verify overhangs: 12" living room, 3" kitchen
- [ ] Install Z-clips or elongated screw method
- [ ] Secure countertop (don't overtighten)

### Finish Countertop

- [ ] Vacuum and tack cloth surface
- [ ] Apply Rubio Monocoat in sections
- [ ] Work product into grain
- [ ] Wipe off ALL excess
- [ ] Buff with clean cloth
- [ ] Let cure: 24-36 hrs light use, 5-7 days full cure

### Quality Gate

Before proceeding to Phase 9:
- [ ] Joint flat and minimal/invisible
- [ ] Surface smooth
- [ ] Countertop level
- [ ] Overhangs correct
- [ ] Pop-up hole clean
- [ ] Countertop secure but can move with humidity
