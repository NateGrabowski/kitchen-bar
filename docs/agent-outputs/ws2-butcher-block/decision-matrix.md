**Runner-up: Home Depot (7.60/10)**

- Buy Lowe's chevron + laminate substrate
- Achieve 2.5" thickness with chevron top
- Total ~$585-685 (competitive with Country Mouldings)
- Get both chevron AND thickness

**Winner: Lowe's (8.45/10)**
- $349 sale price (best value)

- Still cheapest total cost even with lamination
