# Armani Fine Woodworking Butcher Block Analysis

**Updated:** 2026-01-09

## Company Overview

Armani Fine Woodworking is a Colorado-based custom woodworking company specializing in sustainably sourced solid wood butcher block countertops, desktops, and floating shelves. They handcraft each piece and ship nationwide.

## Custom Capabilities

### Size Range
- **End Grain Format:** Up to 45" x 96"
- **Edge Grain Format:** Custom sizes available
- **Thickness:** 1.5" to 7.5" (extraordinary range)

Armani is the **best option IF:**
1. You want an **extraordinary thickness** (3", 4", 5", even 7.5")
2. You prioritize **handcrafted quality** over cost

## Recommended Configuration (If Using Armani)

**For Modern Farmhouse with Maximum Impact:**
- **Species:** Calico Hickory (light sapwood, dramatic grain, perfect farmhouse aesthetic)
- **Size:** 92" L x 30" W x **3" thick** (go thicker if using premium vendor)

## Overall Assessment

**Rating: 6/10 for project fit**

**Strengths:**
- Ultimate quality and craftsmanship
- Extraordinary thickness options (unique selling point)
- Calico Hickory is ideal for modern farmhouse
- Handcrafted in USA with sustainable practices

**Weaknesses:**
- **Cannot provide chevron pattern** (deal-breaker if chevron is required)
- **Premium pricing** likely exceeds budget ($1,000+)
- **Overkill for 2" thickness** (their specialty is 4"-7.5" ultra-thick)
- 3-week lead time

**Recommendation:** Only use Armani if:
1. You abandon the chevron requirement
2. You want to go **3" or thicker** (their specialty)
3. You're willing to spend $1,000-1,600 for handcrafted premium quality

Otherwise, Lowe's chevron at $349 or Country Mouldings at $500-700 are more practical choices.
