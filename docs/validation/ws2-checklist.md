## Scoring Criteria (10 points each)

### 1. Vendor Coverage (0-10)
- [ ] Minimum 5 vendor options researched
- [ ] Each vendor has specific pricing ($ amounts, not "call for quote")
- [ ] Mix of retail (HD, Lowe's) and specialty sources
- [ ] Local options explored if applicable

**Score: ___/10**

### 2. Product Comparison (0-10)
- [ ] Chevron vs plain comparison with pros/cons
- [ ] Thickness options documented (1.5", 2", 2.5"+)
- [ ] Wood species comparison (acacia, maple, oak, etc.)
- [ ] Finish options noted (unfinished, pre-oiled, etc.)

**Score: ___/10**

### 3. Sizing & Joining (0-10)
- [ ] Analysis of single slab vs joining for 92" length
- [ ] If joining needed: method specified (Titebond III, biscuits, countertop bolts)
- [ ] Seam placement recommendation if joining
- [ ] Maximum available sizes per vendor documented

**Score: ___/10**

### 4. Installation Method (0-10)
- [ ] Attachment method researched (countertop bolts, figure-8 fasteners, etc.)
- [ ] Wood movement accommodation addressed
- [ ] Overhang support requirements noted
- [ ] Specific hardware recommendations with sources

**Score: ___/10**

### 5. Decision Matrix & Recommendation (0-10)
- [ ] Clear decision matrix comparing all options
- [ ] Factors: price, availability, delivery time, quality, style match
- [ ] Single clear recommendation with reasoning
- [ ] Know exactly: what to buy, from where, for how much, delivery timeline

**Score: ___/10**
