## Reaching Your 37-41" Total Height

IKEA's standard setup = 30" cabinet + 4.5" legs + 1.5" countertop = **36" total**

You need **37-41"**. Here are your options:

### Option 1: Taller CAPITA Legs + Custom Toe Kick

| Component | Height |
|-----------|--------|
| SEKTION frame | 30" |
| CAPITA legs (4.5") | 4.5" |
| Butcher block | 1.5-2" |
| **Total** | **36-36.5"** |

❌ Still 0.5-4.5" short of your target

### Option 2: Custom 2×4 Base Platform (Recommended)

Build a platform base instead of using IKEA legs:

| Component | Height |
|-----------|--------|
| SEKTION frame | 30" |
| Custom 2×4 platform | **5-9"** (you control this) |
| Butcher block | 1.5-2" |
| **Total** | **36.5-41"** ✅ |

**How to build the platform:**
1. Build a rectangular frame from 2×4s (actual height ~3.5")
2. Stack 2×4s or add plywood to reach desired height
3. For 8" base: 2×4 frame + ¾" plywood top + 3.5" second layer = ~8"
4. Level carefully (use shims if needed)
5. Screw cabinets down through bottom panel

**Pros:**
- Rock solid (more stable than IKEA legs)
- Exact height control
- Can add recessed panel/detail to front for farmhouse look

### Option 3: Extended Toe Kick with Standard Legs

| Component | Height |
|-----------|--------|
| SEKTION frame | 30" |
| IKEA legs | 4.5" |
| Custom toe kick spacer | 2-4" |
| Butcher block | 1.5-2" |
| **Total** | **38-40.5"** ✅ |

Add a wood spacer between the legs and the visible toe kick panel.

## The Living Room Face (Cover Panels)

Since your cabinet faces the living room, you need finished panels on the back.

### IKEA Cover Panel Options

| Style | Size | Price | Match |
|-------|------|-------|-------|
| AXSTAD matte white | 25×30" | $54 | Shaker doors |
| ENKÖPING white wood | 25×30" | $62 | Farmhouse |
| FÖRBÄTTRA matte white | 25×30" | $37 | Budget option |
| ASKERSUND ash | 26×36" | $49 | Natural wood |

**For 92" length × 37-41" height:**
- Need ~3-4 panels to cover the back
- Panels come 30" or 36" tall - may need trimming
- Use IKEA connector rail to attach panels to cabinet backs

### Custom Panel Alternative (Recommended)

Build your own facing panel for a better farmhouse look:
- ¾" plywood base
- Add **shiplap** or **beadboard** overlay
- Or route Shaker-style panels
- Paint to match IKEA doors

This gives you:
- Exact height match (no awkward seams)
- True farmhouse aesthetic
- Full control over style

## Step-by-Step Build Sequence

1. **Order IKEA components** - Check stock, may need store pickup
2. **Build 2×4 base platform** to exact height needed (5-9")
3. **Position and level platform** on kitchen floor at step edge
4. **Assemble SEKTION frames** (without wall brackets)
5. **Set frames on platform**, screw down through bottom
6. **Install shelves and drawer hardware**
7. **Build/install living room facing panel**
8. **Install doors and drawer fronts**
9. **Add open shelving section** (above or integrated)
10. **Install butcher block top**
11. **Add trim, LED lighting, finishing touches**

```
Target total height:        _____ " (37-41")
```
