# Kitchen Bar Height Constraints

**Date:** 2026-01-08
**Purpose:** Quick reference for cabinet and bar height calculations

---

## Fixed Constraints

| Parameter | Value | Notes |
|-----------|-------|-------|
| Step height | **23.5"** | Can't change - it's the floor |
| Kitchen floor to LR floor | 23.5" | Same as step |

---

## Adjustable Range (from Visualization)

| Parameter | Current | Min | Max | Notes |
|-----------|---------|-----|-----|-------|
| Counter height (from kitchen floor) | 38.5" | 36" | 42" | Slider range |
| Butcher block thickness | 1.5" | 1.5" | 2"+ | Flexible |
| Stool height | 28" | 24" | 32" | Counter stool: 24-26", Bar stool: 28-30" |

---

## Target Range

**Total height goal: 37" - 41"** (from kitchen floor, including toe kick + cabinet + countertop)

---

## Height Calculation Formula

```
Total Height = Toe Kick/Base + Cabinet Body + Countertop

Cabinet Body = Total Height - Toe Kick - Countertop
```

## Base Platform Build Heights

Common lumber dimensions for base platform:

| Material | Actual Height | Stacked Options |
|----------|---------------|-----------------|
| 2×4 flat | 1.5" | - |
| 2×4 on edge | 3.5" | - |
| 2×6 on edge | 5.5" | = 37" total (with 30" + 1.5") |
| 2×4 + 2×4 | 7" | = 38.5" total |
| 2×8 on edge | 7.25" | = 38.75" total |
| 2×4 + 2×6 | 9" | = 40.5" total |
| 2×10 on edge | 9.25" | = 40.75" total |

**Tip:** Add ¾" plywood on top of frame for a flat mounting surface.

## From Living Room Perspective

The bar top height from **living room floor**:

```
LR view height = Kitchen counter height + Step height
               = 40" + 23.5"
               = 63.5" (about 5'3")
```

This is why it reads as a "half-wall with storage" from the living room side.
